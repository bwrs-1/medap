# 画面設計 v0.1

## 1. タブ構成（TabView）

```
┌─────────────────────────────────────┐
│  [図鑑] [容器] [設定]               │
└─────────────────────────────────────┘
```

### タブ1: 図鑑（Varieties）
- 品種一覧・検索・詳細

### タブ2: 容器（Containers）
- 容器一覧・詳細・品種紐付け

### タブ3: 設定（Settings）
- ログイン状態・権限表示・ログアウト

---

## 2. 画面一覧

### 2.1 図鑑タブ

#### 2.1.1 品種一覧画面（VarietyListView）
**UI要素**:
- 検索バー（品種名・系統でフィルタ）
- グリッド表示（2列）
  - サムネイル画像（150x150pt）
  - 品種名（18pt Bold）
  - 系統名（14pt Gray）
- 右上に「+」ボタン（Editor以上のみ表示）

**アクセシビリティ**:
- 検索バー: VoiceOver「品種を検索」
- カード: 「{品種名}、{系統}、タップで詳細」

#### 2.1.2 品種詳細画面（VarietyDetailView）
**UI要素**:
- 大きな画像（全幅、300pt高）
- 品種名（24pt Bold）
- 系統（16pt）
- 説明文（16pt、行間1.5）
- 特徴タグ（Capsule形式）
- 飼育難易度（★表示）
- 価格帯（テキスト）
- 編集ボタン（Editor以上）

**アクセシビリティ**:
- 画像: alt「{品種名}の写真」
- 難易度: 「飼育難易度5段階中3」

#### 2.1.3 品種編集画面（VarietyEditView）
**UI要素**:
- Form形式
  - 品種名（TextField）
  - 系統（Picker）
  - 説明（TextEditor、最小5行）
  - 画像URL（TextField）
  - 特徴（複数選択可能なタグ）
  - 難易度（Stepper 1-5）
  - 価格帯（TextField）
- 保存/キャンセルボタン

---

### 2.2 容器タブ

#### 2.2.1 容器一覧画面（ContainerListView）
**UI要素**:
- List形式
  - 容器名（18pt Bold）
  - サイズ・設置場所（14pt Gray）
  - 飼育中の品種数（Badge）
- 右上に「+」ボタン

**アクセシビリティ**:
- 行: 「{容器名}、{サイズ}、{品種数}種類飼育中」

#### 2.2.2 容器詳細画面（ContainerDetailView）
**UI要素**:
- 容器情報（名前・サイズ・場所・メモ）
- 飼育中の品種リスト
  - 品種名・個体数
  - スワイプで削除（Editor以上）
- 「品種を追加」ボタン
- 編集ボタン

#### 2.2.3 容器編集画面（ContainerEditView）
**UI要素**:
- Form形式
  - 容器名（TextField）
  - サイズ（TextField、例: 60cm水槽）
  - 設置場所（TextField）
  - メモ（TextEditor）
- 保存/キャンセルボタン

#### 2.2.4 品種紐付け画面（AddVarietyToContainerView）
**UI要素**:
- 品種選択（Picker、図鑑から選択）
- 個体数（Stepper、初期値1）
- 追加ボタン

---

### 2.3 設定タブ

#### 2.3.1 設定画面（SettingsView）
**UI要素**:
- ユーザー情報セクション
  - 表示名
  - 権限レベル（Viewer/Editor/Admin）
- ログアウトボタン（赤色）
- バージョン情報
- 監査ログ閲覧（Admin のみ）

## 3. 画面遷移図（テキスト）

```
[図鑑タブ]
  ├─ 品種一覧
  │   ├─ タップ → 品種詳細
  │   │   └─ 編集ボタン → 品種編集（Editor以上）
  │   └─ +ボタン → 品種編集（新規）
  │
[容器タブ]
  ├─ 容器一覧
  │   ├─ タップ → 容器詳細
  │   │   ├─ 編集ボタン → 容器編集
  │   │   └─ 品種追加 → 品種紐付け画面
  │   └─ +ボタン → 容器編集（新規）
  │
[設定タブ]
  └─ 設定画面
      └─ ログアウト → ログイン画面
```

## 4. アクセシビリティ要件まとめ

### 高齢者対応
- **最小文字サイズ**: 16pt（見出しは18pt以上）
- **タップ領域**: 最小44x44pt（Appleガイドライン準拠）
- **コントラスト**: 背景白/文字黒（WCAG AA）
- **余白**: 要素間は最低16pt

### VoiceOver
- 全画面で適切なラベル設定
- 画像には代替テキスト
- ボタンは役割を明示（例: "品種を追加"）

### Dynamic Type
- システムフォントサイズに追従
- レイアウトは可変（固定幅を避ける）
